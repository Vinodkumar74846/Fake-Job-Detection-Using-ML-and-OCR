About Job Post Authenticator

Job Post Authenticator** is a web-based application designed to help users identify whether a job posting is real or fake. The system allows users to either upload an image of a job post or paste the job description directly, and it uses Optical Character Recognition (OCR) via Tesseract to extract text from images. The extracted or pasted text is then analyzed using a rule-based approach that scans for suspicious phrases like "Urgent Hiring", "No Experience Needed", "Immediate Hiring", and "Pay to Apply"—all common in fraudulent listings. If any red flags are found, the job post is flagged as fake and reasons are displayed to the user. Built with Python using the Flask framework, this project incorporates HTML and CSS for the frontend and manages user sessions for login, signup, and prediction history. The signup form validates usernames (disallowing numbers), ensures phone numbers are exactly 10 digits, and checks passwords. It also features sample legitimate and fake job examples, a scrolling banner for better user experience, and an "About" section explaining the project. The system can be enhanced further by integrating machine learning models to improve prediction accuracy and report scam jobs to authorities.

Technologies Used:

  • Python (Flask)
  • Pytesseract (OCR)
  • HTML & CSS
  • NLP for text Extraction
  • Ml for Model Training

Rule-Based Detection:

It checks for red flags like:

  • \"Urgent hiring\"
  • \"No experience needed\"
  • \"Pay to apply\"
